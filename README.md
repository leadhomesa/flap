# Leadhome flappy racing
Flappy racing game built with [Phaser](https://phaser.io) for the 31% faster campaign.

## Getting started
```
cp sample.env .env
npm install
```

## Local development with HMR
```
npm start
```

## Production build
```
npm run build
```
